## Что даёт мод

FrogHelper собирает важную игровую информацию в одном месте и показывает её прямо во время игры:

- таймеры боссов, зелий, способностей, продавцов, рун, майнеров и питомцев;
- прогресс уровня и комбо;
- виджеты для рецептов, карты данжа/осады, поп-апов;
- улучшения чата: вкладки, метки времени, копирование сообщений, приглушённый стиль;
- быстрые действия через удержание одной клавиши;
- world-метки, маркеры рыбалки, подсветка полезных объектов;
- локальные уведомления о важных событиях;
- удобный редактор размещения HUD-элементов.

## Быстрый старт

После входа в игру полезно знать четыре вещи:

| Действие | По умолчанию |
| --- | --- |
| Включить режим редактирования HUD | `H` |
| Открыть меню настроек | `O` |
| Открыть Quick Access | удерживать `R` |
| Включить/выключить авто-кликер | `K` |

Также доступны отдельные горячие клавиши:

| Действие | По умолчанию |
| --- | --- |
| `clanhide` | `Z` |
| `runesbag` | `X` |
| Поставить игровую метку | средняя кнопка мыши |

## Основные хоткеи

### `H` — Editing Mode

Открывает режим редактирования HUD. В нём можно:

- перетаскивать виджеты;
- менять их размер;
- привязывать к краям экрана;
- сцеплять виджеты между собой;
- собирать группы связанных элементов.

### `O` — Settings Menu

Открывает внутреннее меню настроек FrogHelper. Через него включаются и настраиваются:

- виджеты;
- чат-улучшения;
- поп-апы;
- рыбалка;
- визуальные эффекты;
- тема интерфейса;
- quick access;
- авто-сообщения;
- clicker и прочие утилиты.

### `R` — Quick Access

Работает по принципу **удержания**:

- пока клавиша удерживается, на экране видно меню быстрых команд;
- когда клавиша отпускается, выполняется команда под наведённым слотом.

### `K` — Clicker

Переключает авто-кликер. Он может:

- имитировать обычную атаку;
- либо использовать предмет вместо атаки;
- работать с настраиваемым CPS.

### `Z` — Clan Hide

Отправляет команду `clanhide`.

### `X` — Runes Bag

Отправляет команду `runesbag`.

### Средняя кнопка мыши

Создаёт и отправляет **игровую метку**. Метка отображается локально у вас и может быть передана другим игрокам через чат-протокол мода.

## Редактор HUD

Редактор HUD нужен для полной подгонки интерфейса под себя.

### Что можно делать

- свободно двигать виджеты;
- привязывать их к экранным якорям;
- сцеплять один виджет с другим;
- переносить целые группы связанных элементов;
- масштабировать отдельные блоки;
- сохранять персональную раскладку.

### Управление в редакторе

| Действие | Как работает |
| --- | --- |
| Перемещение | зажать ЛКМ по виджету |
| Масштабирование | `Ctrl` + колесо мыши |
| Тонкая регулировка масштаба | `Ctrl` + `Shift` + колесо |
| Перетаскивание группы | удерживать `Alt` при начале перетаскивания |
| Привязка к экранному якорю | поднести виджет к якорной точке |
| Сохранение позиции | автоматически после отпускания виджета |

### Что особенно удобно

- виджеты можно выстроить в связки и двигать блоком;
- можно делать компактный интерфейс под PvE, PvP или фарм;
- расположение и масштаб запоминаются между запусками.

## HUD и виджеты

Ниже перечислены основные HUD-элементы, которые мод умеет показывать игроку.

### Боссы и бой

| Виджет | Что показывает |
| --- | --- |
| **Boss Timers** | список боссов с таймерами возрождения |
| **Boss Bar** | отдельный кастомный блок с полосой/статусом босса |
| **Boss Damage** | ваш текущий или последний урон по боссу |
| **Ability Cooldowns** | откаты способностей |
| **Wand Cooldowns** | откаты инструментов/палок/умений этого типа |
| **Combo Progress** | текущий прогресс комбо |
| **Blocks/s** | скорость ломания блоков |

#### Дополнительно по боссам

В настройках можно включить:

- имя босса;
- иконки;
- уровень;
- таймер;
- режим источника таймеров;
- лимит количества отображаемых боссов;
- фильтрацию по минимальному и максимальному уровню.

### Прогресс и активные эффекты

| Виджет | Что показывает |

| **Active Runes** | активные руны |
| **Active Pets** | активных питомцев |

| **Seller Cooldowns** | откаты продавцов/рынка и похожих систем |
| **Visibility Status** | текущий режим видимости клана, игроков и питомцев |

### Рецепты и полезные панели

| Виджет | Что делает |
| --- | --- |
| **Potion Recipe** | показывает рецепт/подсказку по текущему зелью |

| **Dungeon / Siege Map** | отображает карту данжа/осады |

### Визуальные элементы вне списка виджетов

Помимо классических HUD-блоков мод также может показывать:

- полосы HP над игроками;
- world-маркеры рыбалки;
- world-метки/пинги;
- подсветку ингредиентов алхимии;
- подсветку полезных предметов и объектов;
- подсветку декораций в данжах/осадах.

### Вкладки чата

Сообщения автоматически раскладываются по вкладкам:

- общий чат;
- приватные сообщения;
- прочие категории, которые можно отделить по префиксам.

### Метки времени

Каждое сообщение можно снабжать временем в формате `HH:mm:ss`, чтобы проще отслеживать:

- когда пришёл приват;
- когда сработал системный ивент;
- когда было отправлено ваше сообщение или уведомление.

### Приглушённый стиль чата

Опция `tone down chat` делает чат визуально спокойнее:

- убирает визуальный шум;
- сокращает агрессивные декоративные символы;
- делает переписку аккуратнее и легче читаемой.

### Копирование сообщений из чата

Мод умеет копировать текст сообщения:

- убираются временные метки;
- убираются лишние игровые префиксы;
- копируется именно полезный текст сообщения.

После копирования можно получить pop-up уведомление.

### Auto Thanks

Если кто-то активировал глобальный бустер, мод может автоматически отправить `thx`.

### Приватные сообщения и pop-up уведомления

На входящий приват можно получить отдельный pop-up:

- с кратким превью текста;
- без необходимости постоянно держать чат раскрытым.

### Обмен таймерами боссов

Доступна команда для отправки своих таймеров другому игроку:

- мод упаковывает список таймеров;
- отправляет его по частям через сообщения;
- у получателя таймеры импортируются в локальную базу.

### Клановый chat-протокол и пинги

Мод умеет передавать игровые метки через чат:

- вы ставите метку средней кнопкой мыши;
- мод кодирует данные;
- другие игроки с FrogHelper получают тот же пинг у себя.

## Quick Access

Quick Access — это **быстрое секционное меню действий**, открываемое удержанием клавиши.

### Что это даёт

- быстрый доступ к частым командам;
- меньше ручного ввода;
- удобный выбор действий буквально за одно удержание клавиши.

### Как это работает для игрока

1. Зажимаете `R`.
2. На экране появляется меню с секциями и предметными слотами.
3. Наводите курсор на нужную команду.
4. Отпускаете клавишу.
5. Команда выполняется автоматически.

### Стандартные секции

В конфиге по умолчанию уже есть заготовки, например:

- **Common**: магазин, улучшения, руны, коллекции, питомцы;
- **Travel**: спавн, шахта, меню, боссы, рыбалка;
- **Player**: действия по игроку, включая шаблоны с `{player}`.

### Подстановка `{player}`

Некоторые команды могут автоматически подставлять ник игрока под прицелом.

Например:

- просмотр статистики цели;
- трейд с целью;
- другие действия по выбранному игроку.

### Встроенный редактор Quick Access

Через настройки можно открыть отдельный редактор и:

- создавать секции;
- добавлять и удалять слоты;
- менять название;
- менять команду;
- менять иконку предмета;
- задавать `CustomModelData`;
- перетаскивать элементы между секциями.

## Метки, оверлеи и визуальные подсказки

### World Ping / WayPoints

Мод поддерживает world-метки:

- ставятся средней кнопкой мыши;
- видны в мире как визуальные маркеры;
- хранят позицию, локацию и автора;
- фильтруются по серверу, миру и игровой локации;
- автоматически исчезают спустя время.

Для waypoints можно настраивать:

- максимальную дистанцию;
- смещение по высоте;
- показ автора;
- показ имени сущности;
- показ координат/локации;
- показ расстояния.

### Рыбалка

В рыболовном блоке мод умеет:

- показывать рыболовные метки в мире;
- выводить виджет по клёву;
- отображать данные по рыболовным точкам;
- автоматически выдёргивать удочку при срабатывании.

### Алхимия

Подсвечивает ингредиенты алхимии прямо в мире.

### Данжи и осады

Поддерживаются:

- карта данжа/осады;
- подсветка важных декораций и элементов окружения.

### Полосы HP игроков

Мод может рисовать полосы здоровья над игроками:

- с настройкой прозрачности;
- с вертикальным смещением;
- с плавным затуханием по дистанции.

### Подсветка полезных объектов

Есть отдельные рендер-улучшения для выделения полезных предметов и объектов мира, чтобы они читались быстрее на фоне окружения.

## Автоматизация и утилиты

### Auto Messages

Система авто-сообщений позволяет настроить слоты с сообщениями, которые будут отправляться по таймеру.

Для каждого слота задаются:

- текст;
- активность;
- задержка в секундах.

### Clicker

Позволяет включить авто-нажатие с заданной скоростью.

Настройки:

- CPS;
- режим атаки;
- режим использования предмета вместо атаки.

### Auto Fish

Полуавтоматическая рыбалка:

- отслеживает состояние поплавка;
- ждёт заданное количество тиков;
- автоматически делает нужное действие.

### Boss Respawn Messages

Мод умеет локально уведомлять о важных этапах босса:

- сообщение перед возрождением;
- сообщение в момент спавна;
- сообщение о проклятии;
- сообщение о низком HP босса.

### Pop-Ups

Встроенная система всплывающих уведомлений сообщает о событиях вроде:

- копирования чата;
- приватного сообщения;
- появления босса;
- готовности способности;
- готовности wand-умения;
- готовности продавца;
- возвращения майнера;
- окончания отката набора рун;
- окончания зелья;
- окончания бустера.

Дополнительно настраиваются:

- количество одновременно видимых pop-up;
- длительность показа;
- скорость появления;
- скорость исчезновения.

## Команды мода

### `/fhtarget`

Локальный список целей для вспомогательных механик.

Примеры:

```text
/fhtarget add Nickname
/fhtarget remove Nickname
/fhtarget list
```

Что это даёт:

- можно быстро вести список интересующих игроков;
- список сохраняется локально в конфиге мода.

### `/fhshare <ник>`

Отправка ваших таймеров боссов указанному игроку.

Что происходит:

- мод собирает локальные таймеры;
- упаковывает их в специальные сообщения;
- отправляет адресату;
- у получателя таймеры импортируются в его локальный трекер.
